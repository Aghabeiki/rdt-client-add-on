# RDT Client Home Assistant Add-on Repository

This repository contains the **RDT Client** add-on for Home Assistant.

## Add-on: RDT Client

A wrapper for [RDT Client](https://github.com/rogerfar/rdt-client), a Real Debrid Torrent Client.

### Installation

1.  Add this repository to your Home Assistant Add-on Store:
    `https://github.com/Aghabeiki/rdt-client-add-on`
2.  Install **RDT Client** from the store.
